Flaskについてメモ

・ファイル名の付け方
Flask自身と衝突を避けるため、ファイル名にflask.pyは禁止。


・実行時
python -m flask --app ファイル名(拡張子抜き) run
flask --app ファイル名(拡張子抜き) run

アプリケーションを見つける時の振る舞い
ファイル名がapp.pyもしくはwsgi.pyなら --appを使用する必要はない
本番時は非推奨

プログラムの中にrunなどの詳細が入っている場合は
python ファイル名


・デバックモード
有効にするとリクエスト処理中にエラーが起きたらブラウザにエラー情報を表示してくれる
本番時は非推奨


リクエストオブジェクト
from flask import request
HTTP上のリクエストメゾットの属性を扱える
